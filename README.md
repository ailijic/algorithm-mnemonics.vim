features
--------

* utilize [UltiSnips VisualPlaceholders](https://github.com/SirVer/ultisnips/blob/master/doc/UltiSnips.txt#L774-L817)
* parametrized snippets
  * `const type qualifiers` of return values 
  * lambda parameters
